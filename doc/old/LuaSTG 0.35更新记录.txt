======== version 0.32 ========
1.增加了changes.txt以详细记录每个版本的变化。
2.为object增加collision初始属性。
3.在执行版本检查时，不再要求引擎版本和mod版本完全一致，现在只要求前者不低于后者。
4.增加了对joystick的支持。
5.对laser进行了少许优化。
6.降低了laser转化为信仰点的数量。
7.增加了弯曲的激光。
8.增加了一些弹形。
9.单个符卡调试时，boss的初始位置不再是(0,144)，而是在定义boss时所填写的初始position。也就是说现在需要在符卡的一开始增加一个（如果原来没有的话）move to来移动boss到需要的位置。

======== version 0.33 ========
1.菜单界面修改。
2.增加了内建函数ExtractRes。
3.增加了内建函数RegTTF。
4.修改LoadTTF函数，现在可以使用自定义的非英文字体。
5.将系统默认的非英文字体（用于对话、boss名称和符卡名称）改为方正黑体。
6.增加了符卡练习模式。
7.取消了0.32版本中的改动9。
8.修正了3D背景up设置的bug。
9.修改了魔法森林背景的bug。
10.引擎不再兼容0.33版本以下的mod，需要从工程文件重新编译。

======== version 0.34 ========
1.略微提升了sin/cos/tan的效率，并且增加了asin/acos/atan/atan2，新增的函数仍然使用角度制。
2.修改了蓝点的的满倍分数判定。
3.增加了camera setter用于调试3D背景的相机设置。
4.修正了调试模式下代码行号偏移的bug。
5.从编辑器调试时可以按小键盘上的+/-来加快/减慢游戏速度。

======== version 0.35 ========
1.在定义类型的时候，可以指定该类型所属的难度，这样在不同难度下创建单位的时候，会自动选择创建对应难度的类型。
2.增加了表示难度的全局变量difficulty。该变量的值可以在相应的stage group中规定。一般约定Easy为1，Normal为2，Hard为3，Lunatic为4。
3.优化了弯曲激光，现在Sampling属性变为无效，无论填多少，都当成1处理。
4.现在，当弯曲激光的bound属性为true时，在尾部出屏时才会被自动清除。
5.修正了文字输入界面把对勾显示为问号的bug。

======== version 0.35b ========
1.修正了camera_setter的渲染bug。
2.修正了弯曲激光会使游戏崩溃的bug。

======== version 0.35c ========
1.修正了launcher的一些bug。
2.修正了子弹图像的一些bug。
